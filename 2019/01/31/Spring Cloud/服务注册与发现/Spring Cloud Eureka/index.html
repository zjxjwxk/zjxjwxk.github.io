Spring Cloud Eureka

  • 基于 Netflix Eureka 做了二次封装
  • 两个组件组成：
    • Eureka Server 注册中心
      • 作为具有服务注册功能的服务器，是服务注册中心，系统中的其他微服务使用 Eureka Client 连接到Eureka Server ，并维持心跳连接以监控各个微服务是否正常运行。
    • Eureka Client 服务注册
      • 用于简化与服务器的交互，作为轮询负载均衡器，并提供服务的故障切换支持。
